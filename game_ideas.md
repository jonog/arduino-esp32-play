# Game Ideas

1. **Color Tap**
   The screen shows a color word, but the text color may disagree. Tap only when the word matches the color.

2. **Max Says**
   Tiny Simon Says: colored quadrants flash in a pattern, and you repeat it by touch.

3. **Micro Pong**
   One-player Pong with a touch-controlled paddle at the bottom.

4. **Bug Squash**
   Little dots or icons appear randomly; tap them before they vanish.

5. **Reaction Duel**
   Wait for the screen to change, then tap as fast as possible. Add fake-out colors where tapping too early loses.

6. **Tiny Maze Tilt-Free**
   Drag a dot through a maze using touch.

7. **Orbit Defender**
   A planet sits in the center; tap around the edge to rotate a shield and block incoming dots.

Best first build: **Max Says**. It fits the hardware, reuses the color system, and introduces touch input without needing complex graphics.
